# Titan-V1
A lightweight, hand-built model rocket designed for low-altitude flights (~100–160m) using a custom paper motor casing and simple recovery system.

📸 Overview

This project is a fully DIY rocket build including:


Hand-wrapped paper motor casing

3D printed nose cone and tail section

Cardboard body tube

Simple recovery system (parachute or streamer)

Built with accessibility in mind — using common materials and basic tools.


📏 Specifications

Component	Value

Body Diameter	25 mm

Total Length	~32–36 cm

Motor Size	18 mm (DIY casing)

Dry Weight	56 g

Flight Weight	~75–80 g

Estimated Altitude	100–160 m

Peak Speed	~35–60 m/s (80–130 mph)

Time to Apogee	~3–5 seconds


🧱 Build Details

🟤 Body Tube
Material: Cardboard

Length: ~22 cm (cut down from 37 cm)

Cut using scoring method for clean edges


🟡 Nose Cone
Material: PLA (3D printed)

Weight: 7 g

drill two holes to embedded metal hook (made from coat hanger) for recovery system


🔵 Tail Section

Material: PLA (3D printed)

Weight: 19 g

Integrated fins for stability

Glue into body using epoxy or strong PVA


⚙️ Motor Casing

Inner diameter: ~18 mm


Built using:

Printer paper (primary layers)

Optional outer reinforcement (thin card)

Constructed via tight layered wrapping (not papier-mâché)


Glue Mix:
70% PVA / 30% warm water


Wrapping Method:

3–4 cm wide strips

Full-length wraps (~90 mm then cut to size)

Majority straight wraps + optional outer spiral layer for outer strength 


🪂 Recovery System

Setup:

Nose cone → string → parachute → body attachment

Recommended:

Parachute diameter: 30–35 cm

Alternative: streamer (60–100 cm long)


Materials:
Plastic bag parachute

Nylon or polyester cord

Notes:

Shock cord length: 60–90 cm

Nose cone should be friction fit snug, not tight


🚀 Flight Characteristics

Expected Performance (not real world tests this project is experimental):

Boost time: ~1 second

Apogee: ~3–5 seconds

Total flight time: ~10–15 seconds

Descent Speeds:


System	Speed

None	45–70 mph

Streamer	15–30 mph

Parachute	8–15 mph


🌬️ Drift Estimates

Wind Speed	Drift (Parachute)

Light (~5 mph)	20–50 m

Medium (~10 mph)	50–120 m

Strong (15+ mph)	100–200 m


⚠️ Important Notes

Balance is critical: ensure centre of mass is forward

PLA components may soften under heat → consider shielding near motor using tin foil

Always test fit and alignment before final gluing

Recovery system strongly recommended to prevent damage
